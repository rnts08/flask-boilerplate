Introduction

I got tired of copying and pasting my corner-stone files every time i was going to write a single flask application, so I generalized the code and created flask-boilerplate. I've seen numours boilerplats and foundations on github but all of them contained to much complexity, I just wanted the basics. I don't want Blueprints or Tests.

I've included the basic features and some Howto's for commonly used solutions, such as passing simple functions to Jinja, for example, you are visiting from {{ client_ip() }} and a nicely formatted now() that can be used in your templates look likes like this: {{ timestamp() }}.
